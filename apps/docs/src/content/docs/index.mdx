## Install in one paste

Paste one block into a root shell on a Linux host with Docker. It asks for a panel URL, an admin
account, and (optionally) SMTP — then clones the Evodactyl source, builds the panel image, brings
the stack up, runs migrations, seeds default eggs, and creates your first admin user.
